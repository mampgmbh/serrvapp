# serrv

**Preview your website on your Mac before it goes live.**

serrv serves your local website files over HTTPS with a `.local` domain — 
so you can test exactly how your site behaves in production, right on your Mac.
No configuration, no Docker, no cloud uploads.

→ **[mamp.info/serrv](https://mamp.info/serrv)**
